# Phone Number Extractor
Mokuton is a tool created to extract information from phone numbers
# Generated information
- Basic info
- International format
- National format
- The number is valid
- Can be contacted internationally
- Location
- Location area code
- Number type
- Certain operators
- ISP
- Time zone
- Geographic number
# Tested on
Termux

Python
